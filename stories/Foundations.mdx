## Foundation Elements

### Colors
Our color system provides semantic colors for consistent UI development:
- **Primary** - Purple accent colors for key actions
- **Secondary** - Blue tones for supporting elements
- **Success/Warning/Error** - Status communication colors
- **Neutral** - Gray scale for backgrounds and text

### Typography
Consistent type scale using Inter and Fira Code:
- **Headings** - Bold, impactful display text
- **Body** - Readable content typography
- **Code** - Monospace for technical content
- **Captions** - Small supporting text

### Spacing
8px grid system for consistent spacing:
- **xs**: 4px
- **sm**: 8px
- **md**: 16px
- **lg**: 24px
- **xl**: 32px
- **2xl**: 48px

### Layout
Flexible grid and container system:
- **Container** - Responsive max-width wrapper
- **Grid** - 12-column responsive grid
- **Flexbox utilities** - Quick layout helpers
- **Spacing utilities** - Margin and padding classes
